Constructors

InfluxDBClient()

Initializes a new instance of the client using connection string.

Supported parameters are:

  • INFLUX_HOST - authentication token (required)
  • INFLUX_TOKEN - authentication token (required)
  • INFLUX_ORG - organization name
  • INFLUX_DATABASE - database (bucket) name
  • INFLUX_PRECISION - timestamp precision when writing data (ns (default), us, ms, s)
  • INFLUX_GZIP_THRESHOLD - threshold for gzipping data when writing (default is 1000)

InfluxDBClient(String)

Initializes a new instance of the client using connection string.

Supported parameters are:

  • token - authentication token (required)
  • org - organization name
  • database - database (bucket) name
  • precision - timestamp precision when writing data (ns (default), us, ms, s)
  • gzipThreshold - threshold for gzip data when writing (default is 1000)
